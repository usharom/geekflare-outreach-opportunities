# geekflare-outreach-opportunities
Tool to find if there are backlinking opportunities for Geekflare AI or API on a given list of websites. You have to give an OpenAI API key and a list of sites. It will scan the sites and give you a list of URLs where there are placement opportunities. (This is only working for Geekflare Connect and Geekflare API as on March 2, 2026.)
